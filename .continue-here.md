<decisions_made>

- Pitch CV uses MIDI pitch bend (not retrigger): bend = (currentPitch - openPitch) * (8191/24); reference anchored to note at gate-open
- Pitch bend range sent via RPN 0 on every JOY note-on (6 extra CC messages per note-on) — necessary for correct range in strict MIDI synths
- Gate-close stillness threshold: axis must be < 15% of joystick threshold (near rest), sustained for 1 full second
- Filter CC params added to APVTS (not addParameter) for simplicity; they save/restore harmlessly as last CC position
- Threshold slider hidden (addChildComponent) not removed — APVTS attachment kept, default value active
- Knob row: NoTextBox to match FREE BPM knob visual size; row height 74px (14 label + 60 knob body)
</decisions_made>

<blockers>

- VST3 auto-install post-build step fails without elevation — manual copy required every build
- Pitch bend range RPN only works in synths that respect it; older/simpler synths may have fixed ±2 semitone range (pitch CV would be limited to ±2 semitones in that case)
</blockers>
