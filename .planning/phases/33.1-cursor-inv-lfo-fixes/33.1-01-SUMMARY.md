## Accomplishments

- Cursor now renders at pad center (getWidth()*0.5f, getHeight()*0.5f) on plugin open — no more (0,0) flash
- Spring return is critically damped (kDamping 0.72 → 0.90): settles in ~150ms with zero overshoot
- LFO rate stick modulation is perceptually symmetric: offset applied in normalized log space (skew=0.35) with +/-0.5 range
- INV mode: Quantizer X/Y labels, Modulation LEFT X/LEFT Y labels, and all 4 filter attachments (mode combos + atten knobs) all swap when INV toggles
- Piano keyboard: black key areas reliably register black key clicks via two-pass hit test
- Battery icon: 3 vertical stripe blocks — green (full), orange (medium), red (low), gray (empty), cyan (wired), "?" (unknown) — replaces imprecise fill bar
- Release build succeeded with 0 errors; VST3 auto-installed by CMake post-build hook

- Attachment swap guarded by `prevInvState_` comparison — fires once on toggle, not 30x/sec
- Battery EMPTY shows 0 lit stripes (all gray) — clearer than a tiny red sliver; outline+nib still drawn for visual reference
- GamepadInput.cpp unchanged — battery redesign is purely paint-side
