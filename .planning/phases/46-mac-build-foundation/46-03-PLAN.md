  <name>Task 1: Build Release with xcodebuild ONLY_ACTIVE_ARCH=NO</name>
  <files></files>
  <action>
Run the xcodebuild Release build from the project root. This will take approximately 15-20 minutes on first build (JUCE and SDL2 compile from source). ONLY_ACTIVE_ARCH=NO is required — it forces Xcode to build all architectures declared in CMAKE_OSX_ARCHITECTURES regardless of the "active" host arch.

```bash
cd /Users/student/Documents/Dimea_Arcade_Chord_Control
xcodebuild \
  -project build-mac/ChordJoystick.xcodeproj \
  -scheme ChordJoystick_All \
  -configuration Release \
  ONLY_ACTIVE_ARCH=NO \
  2>&1 | tee /tmp/xcodebuild-46.log | tail -30
echo "xcodebuild exit: $?"
```

If xcodebuild fails (exit non-zero):
- Review the last 50 lines of /tmp/xcodebuild-46.log
- Look for: linker errors (architecture mismatch = CMAKE_OSX_ARCHITECTURES not set before FetchContent), compile errors (C++ syntax from ParameterID changes)
- If ParameterID syntax error: fix the specific line in PluginProcessor.cpp and rebuild
- If arch mismatch linker error: confirm cmake configure was run fresh (build-mac/ wiped before Plan 01)
