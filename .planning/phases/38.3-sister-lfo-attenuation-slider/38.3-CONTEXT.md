# Phase 38.3: Sister LFO Attenuation Inline Slider

## Goal
When a Sister LFO cross-modulation target is selected, shrink the Sister combo to half-width and show a bipolar attenuator slider in the freed right half. When "None" is selected the combo stays full-width and the slider is hidden.

## Background
The Sister LFO combos (lfoXSisterBox_ / lfoYSisterBox_) let you select a cross-modulation target (e.g. LFO-Y modulates LFO-X's frequency). Currently there is no way to control the depth of that modulation without going through a separate knob. An inline attenuator directly beside the source selector makes the relationship obvious and saves vertical space.

## Decisions

### A — Layout behaviour
- **"None" selected:** combo fills full row width (status quo)
- **Any other option selected:** combo shrinks to ~50% of row width; a Slider appears in the right ~50%
- Row height stays 22px (same as the combo) — slider is horizontal, thumb-style

### B — Attenuator range
- **Bipolar: −1.0 .. +1.0** (centre = 0, no modulation; positive = in-phase; negative = inverted)
- Default: +1.0 (full depth, maintains current behaviour on first load)

### C — Dropdown label rename
The Sister combo options are renamed to shorter labels so they fit in the halved combo width:
- In **LFO X box** (options are Y-sourced): "None", "Y Freq", "Y Phase", "Y Level"
- In **LFO Y box** (options are X-sourced): "None", "X Freq", "X Phase", "X Level"

Existing option indices must not change — only the display strings update.

### D — Which dropdowns
Both Sister combos:
- `lfoXSisterBox_` (in LFO X panel)
- `lfoYSisterBox_` (in LFO Y panel)

### E — APVTS storage
Two new AudioParameterFloat params:
- `lfoXSisterAtten` (−1.0 .. +1.0, default +1.0)
- `lfoYSisterAtten` (−1.0 .. +1.0, default +1.0)

The processor already reads the Sister combo to determine cross-modulation target — it multiplies the sister LFO output value by the attenuator before applying it.

## Key Files
- `Source/PluginEditor.h` — 2 new juce::Slider members: `lfoXSisterAttenSlider_`, `lfoYSisterAttenSlider_`
- `Source/PluginEditor.cpp` — combo population (rename labels), onChange show/hide + resized() row-split, timerCallback sync
- `Source/PluginProcessor.cpp` — 2 new APVTS params, attenuator applied at cross-mod injection site

## Constraints
- Row height stays 22px — no new vertical space consumed
- Slider uses same colour scheme as existing sliders (Clr::accent / Clr::panel)
- addChildComponent (not addAndMakeVisible) for sliders that start hidden — lesson from Phase 38.2
- Existing Sister combo selectedId indices must not shift

## Deferred
- Per-voice Sister attenuation (one global atten per LFO axis is sufficient for v1.9)
