  <how-to-verify>
Setup required before testing:
1. Install the freshly built VST3 from the build output directory into your DAW's VST3 scan folder (or load directly if your DAW supports it)
2. Open loopMIDI, create a virtual MIDI port named "CJ-Monitor"
3. Open MIDI-OX, set input to "CJ-Monitor"
4. In your DAW (Ableton or other VST3 host), route the ChordJoystick MIDI output to the "CJ-Monitor" virtual port
5. Connect your PS/Xbox gamepad

--- PATCH-01: CC64=127 sustain-on before note-on ---
- Press any touchplate (or trigger a note via gamepad L1-R2)
- In MIDI-OX: every note-on event must be immediately preceded by CC64=127 on the same channel
- PASS if: CC64=127 appears on the same channel tick before each NoteOn
- FAIL if: NoteOn appears without a preceding CC64=127

Also test looper path:
- Arm REC, press a pad, stop recording, press PLAY
- MIDI-OX: looper note-on playback must also show CC64=127 before each NoteOn
- PASS if: looper-triggered NoteOn events are also preceded by CC64=127

--- PATCH-02: Hold mode inverts gate ---
- Enable HOLD on a voice (the HOLD button in the plugin UI)
- With a synth loaded and held note configured: press the touchplate for that voice
- PASS if: press silences the note (note-off fired), release resumes the note (note-on fired)
- FAIL if: press sounds the note (normal behavior — inversion not working)

--- PATCH-03: Filter Mod gates CC and greys UI ---
- Turn Filter Mod OFF in the plugin UI
- Move the gamepad left stick
- PASS if: zero CC74/CC71 events appear in MIDI-OX; filter dropdowns and knobs are visually dimmed in the plugin UI
- FAIL if: CC events appear with Filter Mod OFF, or UI controls remain active/bright

--- PATCH-04: Panic is one-shot ---
- Hold a note (trigger a gate and leave it active)
- Press MIDI PANIC button
- PASS if: allNotesOff burst appears in MIDI-OX, button immediately returns to "MIDI PANIC" label (not "MUTED"), and the next note trigger produces a NoteOn (no output blocking)
- FAIL if: button shows "MUTED", subsequent triggers produce no MIDI output until pressed again

--- PATCH-05: Filter looper records left-stick gestures ---
- Connect gamepad, Filter Mod ON
- Press REC in the looper section
- Move the left stick (sweep cutoff/resonance)
- Press REC again to stop recording
- Press PLAY
- In MIDI-OX during playback: filter CC events (CC74 and/or CC71) should replay matching the recorded gesture
- PASS if: CC74/CC71 events appear during looper playback even without touching the stick
- FAIL if: only gate events play back, no filter CC in the loop

--- PATCH-06: Atten knob change does not retrigger filter CC ---
- Connect gamepad, Filter Mod ON, left stick held stationary (centred)
- In MIDI-OX, observe the CC stream
- Wiggle the "Filter Cutoff Atten" knob up and down
- Wiggle the "Filter Res Atten" knob up and down
- PASS if: zero new CC74/CC71 events appear in MIDI-OX while the stick is still
- FAIL if: CC74 or CC71 events appear when only the atten knob moved (stick did not move)
