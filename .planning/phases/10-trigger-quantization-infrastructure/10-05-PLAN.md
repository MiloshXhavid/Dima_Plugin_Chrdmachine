<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
    Complete trigger quantization feature:
    - Plan 01: snapToGrid pure function + TC 12 Catch2 test (wrap edge case)
    - Plan 02: APVTS params (quantizeMode, quantizeSubdiv) + processor passthrough API
    - Plan 03: LooperEngine backend — live snap in recordGate(), post-record flag + shadow copy
    - Plan 04: UI — Off/Live/Post buttons + subdivision dropdown in Looper section
  </what-built>
  <how-to-verify>
    Load the plugin in your DAW. Run through this checklist:

    **1. UI visibility**
    - Open the plugin editor
    - Confirm: three buttons [Off] [Live] [Post] visible inside the Looper section
    - Confirm: subdivision dropdown showing "1/8" by default, next to the mode buttons
    - Confirm: only one mode button lit at a time (pressing Live turns off Off)

    **2. Live quantize**
    - Set mode to "Live", subdivision to "1/8"
    - Start the looper (PLAY), arm recording (REC), play a few notes on pad voices at irregular timing
    - Stop recording
    - Play back — gate events should sound rhythmically tighter, on or near 1/8-note boundaries
    - Set mode to "Off", re-record the same irregular presses — play back sounds rhythmically loose (unquantized)

    **3. Post quantize**
    - Record a loop with irregular timing (mode = Off or Live)
    - After recording, set mode to "Post"
    - Play back — events should snap to the nearest 1/8-note boundary
    - Switch mode back to "Off" — events should revert to their original unquantized positions
    - Confirm: switching Post on/off can be done during playback (controls not greyed during playback)

    **4. Disable during recording**
    - Start recording (REC button active)
    - Confirm: all four quantize controls (Off/Live/Post buttons + subdivision dropdown) are greyed/disabled
    - Stop recording
    - Confirm: controls re-enable

    **5. Subdivision selector**
    - Change to "1/16" subdivision with Live mode
    - Record a loop — events should snap to 1/16-note boundaries (tighter grid than 1/8)
    - Change to "1/4" — events should snap to quarter-note boundaries (looser grid)

    **6. Session persistence**
    - Set mode to "Live", subdivision to "1/16"
    - Save the DAW session
    - Reload the session
    - Confirm: mode is still "Live" and subdivision is still "1/16"

    **7. MIDI monitor check (optional but recommended)**
    - Use a MIDI monitor (MIDI-OX, LiveMon, or DAW MIDI track view)
    - With Live mode + 1/8 subdivision: record gates while watching the monitor
    - Note events should align to positions that are multiples of 0.5 beats from the loop start
  </how-to-verify>
