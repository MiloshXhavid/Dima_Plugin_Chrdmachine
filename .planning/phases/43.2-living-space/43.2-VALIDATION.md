## Manual-Only Verifications

| Behavior | Requirement | Why Manual | Test Instructions |
|----------|-------------|------------|-------------------|
| Stars drift in unified heading, joystick banking rotates clockwise | LIVE-01 | Real-time visual, no automated oracle | Load plugin; move joystick right while watching starfield heading rotate |
| Heading persists after joystick release (no snap-back) | LIVE-02 | Visual state persistence | Steer to ~45°, release joystick, confirm stars continue at new heading |
| Stars twinkle independently ±10% brightness | LIVE-03 | Perceptual effect | Observe idle pad for 5+ seconds; brightness variation should be felt, not jarring |
| Shooting star appears every ~20s (random interval 12–30s) | LIVE-04 | Stochastic timing | Observe for 60+ seconds; at least 2–3 streaks should appear |
| Three parallax speed layers felt clearly | LIVE-05 | Visual depth perception | Observe: large bright stars move fastest, tiny far stars move slowest |
| 2–3 nebula blobs visible, drifting | LIVE-06 | Visual presence | Soft purple/teal blobs should be visible at ~5% alpha below starfield |
| Background hue shifts slowly (±8° over 120s) | LIVE-07 | Very subtle, long-duration | Observe background color over 30+ seconds; slight warm/cool oscillation |
| All effects at full strength when looper stopped | LIVE-08 | Warp crossfade boundary | Confirm no dimming while looper is idle |
| If Phase 42 present: nebulae + twinkle + shooting star fade with warpT | LIVE-09 | Forward-compat guard | Start looper playback (if Phase 42 re-implemented); effects should dissolve as warp ramps in |
