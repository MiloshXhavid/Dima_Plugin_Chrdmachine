    9. Set Gate Length knob to ~50% — confirm each arp note occupies roughly half its subdivision slot (short gap before next note-on)

    14. Repeat 2–3 times to confirm consistency

    16. Disable ARP — confirm all notes stop immediately
    17. Re-enable ARP — confirm the first note is voice 0 (root), not a continuation from mid-sequence
